# study-planner
A web application to help myself plan and track my university studies and credits

## Goals
-Help me manage university coursework.
-Keep class materials organized.
-Track degree progress and credits.

## Planned Features

### Core Features
- Calendar-based study planner
- Weekly timetable management
- Study task tracking

### Future Features
- PDF storage for slides and notes
- Homework and revision tracking
- Credit tracker
